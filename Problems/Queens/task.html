Queens Open on JetBrains Academy

There are coordinates of two queens on a chessboard. Find out, whether they take each other.

Input data format

Four integer numbers \( x_1, y_1, x_2, y_2 \).

Output data format

Type "YES" (uppercase) if they take each other and "NO" if they don't.

You may need a method, that calculates the absolute value of the number. Look, it is here:

Math.abs(n)

Sample Input:
1 1 3 3
Sample Output:
YES

Sample Input:
1 1 4 3
Sample Output:
NO

Sample Input:
2 2 5 2
Sample Output:
YES


Memory limit: 256 MB
Time limit: 5 seconds
