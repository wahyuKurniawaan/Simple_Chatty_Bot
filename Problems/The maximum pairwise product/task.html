The maximum pairwise product Open on JetBrains Academy

Write a program that reads N numbers and outputs the maximum pairwise product of the given N numbers.

If there is only one element, output this element.

The first line of the input contains the number N.
The second line contains N numbers separated by spaces. All the numbers are positive.


Sample Input:
2
5 3
Sample Output:
15

Sample Input:
3
10 2 4
Sample Output:
40


Memory limit: 256 MB
Time limit: 8 seconds
